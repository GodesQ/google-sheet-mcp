üõ†Ô∏è Available Operations

Main Tool: manage-sheet

This tool handles all CRUD operations with the following parameters:

Parameters:

  • business_sector_type: One of the supported sectors above
  • operation: "add", "update", "delete", or "read"
  • newRow: Object with column-value pairs (for add operations)
  • rowIndex: 1-based row index (for update/delete operations)
  • cellUpdates: Array of column-value updates (for update operations)
  • select: Array of columns to include (for read operations)
  • filter: Array of filter conditions (for read operations)
  • limit: Maximum number of rows to return (for read operations)
  • offset: Number of rows to skip (for read operations)

Example Usage:

// Read invoices with pagination { "business_sector_type": "invoices", "operation": "read", "limit": 10, "offset": 0 } // Add a new invoice { "business_sector_type": "invoices", "operation": "add", "newRow": { "Invoice Number": "INV-001", "Client": "John Doe", "Amount": "1000.00", "Status": "Pending" } } // Update an invoice { "business_sector_type": "invoices", "operation": "update", "rowIndex": 2, "cellUpdates": [ {"column": "Status", "value": "Paid"}, {"column": "Amount", "value": "1200.00"} ] }

üîß Setup Requirements
